Learn more about our self-healing web application and its features.

🎯 Project Overview

This self-healing web application demonstrates modern DevOps practices including containerization, CI/CD pipelines, and automated recovery mechanisms. Built with Node.js, Express, and Docker, it showcases how applications can maintain high availability through intelligent restart policies.

🛠️ Technologies Used

Backend

  • Node.js
  • Express.js
  • RESTful APIs

Frontend

  • HTML5
  • CSS3 (Glassmorphism)
  • JavaScript (ES6+)

DevOps

  • Docker
  • Jenkins CI/CD
  • Health Monitoring

✨ Key Features

  • Self-Healing: Automatic recovery from crashes using Docker restart policies
  • Containerized: Fully dockerized for consistent environments
  • CI/CD Pipeline: Automated testing and deployment with Jenkins
  • Health Monitoring: Built-in health checks and monitoring
  • Interactive Demo: Counter feature demonstrating state management
  • Responsive Design: Works on all device sizes

🚀 Getting Started

1. Clone the Repository

git clone https://github.com/your-repo/self-healing-app.git

2. Install Dependencies

cd app && npm install

3. Run Locally

npm start

4. Build with Docker

docker build -t self-healing-app .
